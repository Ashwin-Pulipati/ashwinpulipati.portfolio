## üìù About The Project

This project is a modern, responsive, and interactive personal portfolio. It's designed to provide a comprehensive overview of my professional profile, including my expertise, work experience, and personal projects. It also includes a direct way to get in touch with me for collaboration or hiring opportunities.

This project is built with a modern tech stack that ensures a great developer experience, scalability, and a beautiful user interface.

*   **Framework:** [Next.js](https://nextjs.org/)
*   **Language:** [TypeScript](https://www.typescriptlang.org/)

*   **Email Service:** [EmailJS](https://www.emailjs.com/)

## ‚úÖ Key Features

- **Responsive Design:** Fully responsive layout that looks great on all devices, from mobile phones to desktops.
- **Interactive UI:** Engaging user interface with animations and interactive elements.
- **Dark Mode:** Theme toggle for a comfortable viewing experience in low-light environments.
- **Detailed Sections:**
    - **About Me:** A personal introduction and my professional pillars.
    - **Expertise:** A breakdown of my skills, tools, and capabilities.
    - **Experience:** An interactive timeline of my professional journey.
    - **Work:** A gallery of my projects.
    - **Hire Me:** A dedicated section for collaboration and job opportunities, including a contact form and Calendly integration.
- **Contact Form:** A functional contact form using EmailJS to send messages directly from the site.

## ‚ñ∂Ô∏è Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have Node.js and npm installed on your machine.
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/Ashwin-Pulipati/ashwinpulipati.portfolio.git
    cd ashwinpulipati.portfolio
    ```
2.  **Install NPM packages:**
    ```sh
    npm install
    ```
3.  **Set up your environment variables:**
    Create a `.env.local` file in the root of your project and add your EmailJS credentials. These are required for the contact form to work.
    ```env
    # EmailJS
    NEXT_PUBLIC_EMAILJS_SERVICE_ID=your_emailjs_service_id
    NEXT_PUBLIC_EMAILJS_TEMPLATE_ID=your_emailjs_template_id
    NEXT_PUBLIC_EMAILJS_PUBLIC_KEY=your_emailjs_public_key
    ```
4.  **Run the development server:**
    ```sh
    npm run dev
    ```
5.  Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.
